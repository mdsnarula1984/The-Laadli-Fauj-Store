# ⚔️ The Laadli Fauj Store
### **Ancient Heritage. Modern Craftsmanship.**

The **Laadli Fauj Store** is a high-performance, boutique e-commerce experience built for the modern Gursikh. It merges the raw, warrior aesthetic of the Khalsa with a minimalist, luxury digital interface. 

---

## 🚀 Performance USP
Built with a "Zero-Lag" philosophy. Every animation, image, and transition is optimized to ensure a smooth experience even on lower-end hardware.
* **No Heavy Blurs:** Optimized CSS gradients for cinematic depth.
* **Lazy Loading:** Images only load when they enter the warrior's view.
* **Static Data:** Blazing fast navigation via a centralized `data.ts` architecture.

## 🛠️ The Arsenal (Tech Stack)
* **Framework:** Next.js 14+ (App Router)
* **Styling:** Tailwind CSS (Custom Modern Theme)
* **Payments:** Stripe API (Akal Sahai Secure Checkout)
* **State:** React Context API (The "Magic Pocket" Cart System)

## ✨ Key Features
* **The ਹਾਟ (The Haat):** A centered, symmetrical product gallery.
* **Quick-Add:** One-tap arsenal building without leaving the gallery.
* **Tyari Bar Tyari:** A custom Nihang-themed order progress tracker.
* **Mobile Responsive:** Designed to look premium on every device.

## 📦 Setup for the Empire
1. **Clone the repository:**
   ```bash
   git clone https://github.com/mdsnarula/The-Laadli-Fauj-Store.git
   ```
2. **Install the gear:**
   ```bash
   npm install
   ```
3. **Configure the Secret Box (.env.local):**
   ```bash
   NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_key
   STRIPE_SECRET_KEY=your_secret
   ```
4. **Ignite the engine:**
   ```bash
   npm run dev
   ```
